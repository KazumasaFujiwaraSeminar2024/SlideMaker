# マーカー一覧

<h2>背景</h2>

```tex
%背景
```

<details close><summary>説明</summary>

背景が書かれているセクションにこのマーカーを書いてください。

例

```tex
%背景
\begin{abstract}
内容
\end{abstract}
```

</details>
