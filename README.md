# Slide Maker

## 説明

このプロンプトは、卒業論文をわかりやすくまとめたスライドを作るためのものです。

論文はTeX（テフ）という形式のファイルで書かれています。

このプロンプトでは、そのTeXファイルから「概要」「目的」「証明」など、
論文の大事な部分を自動で取り出し、スライドの形に整理します。
